# AquaMind AI

AI-Powered Smart Water Management & Conservation System

## Features
- **Dashboard**: Track daily, weekly, monthly water usage.
- **AI Prediction**: Predict future usage using Random Forest.
- **Leak Detection**: Detect abnormal usage using Isolation Forest.
- **Recommendations**: Get personalized water-saving tips.
- **Water Bill Estimator**: Calculate estimated bills based on usage.
- **Goals**: Set and track water-saving goals.

## Architecture
- **Frontend**: React, Vite, Tailwind CSS, Recharts
- **Backend**: FastAPI, SQLAlchemy
- **Database**: MySQL
- **ML**: Scikit-Learn
